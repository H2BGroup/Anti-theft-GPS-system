## Config file
For the program to work a `config.json` file in main directory is needed. The content should be json formatted:
```json
{
    "device_secret": "",
    "owner_number": "",
    "device_number": "",
    "rabbit_host": "",
    "rabbit_user": "",
    "rabbit_password": ""
}
```
